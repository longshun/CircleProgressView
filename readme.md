**属性：**
- 1.设置进度条的最大值max；max
- 2.设置当前加载进度progress;
- 3.设置是否显示中间的百分比文字；textIsNeedShow
- 4.设置文字尺寸，颜色；textSize，textColor
- 5.设置圆环颜色；circleColor
- 6.设置加载进度圆环颜色；circleProgressColor
- 7.设置圆环宽度；circleWidth
